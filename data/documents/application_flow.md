# 申請流程

貸款申請流程包含資料輸入、系統欄位檢查、模型風險預測、風險分級、人工覆核與結果紀錄。系統會保存每次預測的輸入資料、預測機率、分級結果、案件編號與時間戳記，以支援後續分析與模型再訓練。

- 申請人提交基本資料、收入資料、貸款資料與信用資料。
- 系統檢查必要欄位是否完整。
- 資料完整後，模型產生違約風險機率。
- 系統依門檻產生低風險、中風險或高風險結果。
- 中風險案件進入人工覆核，高風險案件需提高審核層級。

# 補件流程

若資料不足或欄位缺漏，審核流程應先暫停並要求補件。補件資料可包含收入證明、工作證明、薪資轉帳紀錄、貸款用途說明、既有負債資料或信用紀錄補充文件。

- 補件完成後才可重新送入模型預測或人工覆核。
- 補件紀錄應與案件編號關聯保存。
- 若補件後仍無法確認還款能力，應維持人工覆核或提高審核層級。

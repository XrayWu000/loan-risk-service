# Loan Approval Classification (Machine Learning)

## 專題簡介
本專題以 **Loan Approval Classification（貸款核准分類）** 為主題，
透過監督式機器學習（Supervised Machine Learning）模型，
根據申請人之個人與財務特徵，預測貸款是否通過核准。

## Project Objective
The objective of this project is to build and evaluate machine learning models
that can classify whether a loan application will be approved or rejected
based on applicant-related features.

## Dataset
本專題使用貸款申請相關資料，可能包含以下特徵：
- Applicant Income（申請人收入）
- Co-applicant Income（共同申請人收入）
- Loan Amount（貸款金額）
- Credit History（信用紀錄）
- Employment Status（就業狀態）
- Property Area（房產區域）

資料將進行：
- 資料清洗（Data Cleaning）
- 特徵工程（Feature Engineering）
- 模型訓練與評估（Model Training & Evaluation）
